Jarvis Virtual Assistant Documentation

1.Overview

Jarvis is a Python-based virtual assistant designed to perform various tasks like web browsing, sending emails, fetching news, telling jokes, checking the weather, and more. It uses multiple APIs and libraries to provide a seamless user experience.

This documentation explains the features, setup, and usage of the Jarvis assistant.

2.Features

3.How it works

core functions

  1. speak(text)

    • Uses pyttsx3 for text-to-speech.

    • Configures voice, speed, and volume.

  2. wish()

    • Greets the user based on the time of day.

  3. web()

    • Opens a website (e.g., open facebook → https://www.facebook.com).

  4. search()

    • Searches YouTube (e.g., search Python tutorials).

  5. news()

    • Fetches headlines from NewsAPI.

  6. wiki()

    • Gets Wikipedia summaries (with error handling).

  7. time()

    • Speaks the current time.

  8. email()

    • Uses OpenAI to draft emails and sends them via SMTP.

  9. weather()

    • Provides weather data using OpenWeatherMap.

  10. hijri()

    • Converts Gregorian date to Hijri.

  11. img()

    • Uses Selenium to generate AI images via Craiyon.

  12. joke()

    • Tells random programming jokes using pyjokes.

  13. ai()

    • Uses OpenAI (via OpenRouter) for general queries.

  14. location()

    • Uses ip address for location.

6.Error Handling

  • Wikipedia Errors → Falls back to Google search.
  • API Failures→ Provides error messages.
  • General Exceptions→ Provides error messages.
